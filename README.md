\# MMO Prototype — Unity (C#)



A server-authoritative MMO prototype built in Unity 2022.3 LTS

using Netcode for GameObjects, demonstrating core MMO engineering concepts.



\## Features



\- Server-authoritative movement with client-side prediction

\- Input reconciliation (client corrects mispredictions)

\- Tick-based game loop at 20Hz

\- Remote player interpolation (no jitter)

\- NetworkVariable synced player stats (HP/MP/Level)

\- Full combat system: attack, damage, death, respawn

\- Kill feed

\- Player nameplates with live HP bars

\- Third-person camera follow
